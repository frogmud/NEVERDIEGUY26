Guidelines

Best practices and conventions for working with the NEVER DIE GUY design system.

Naming Conventions

File Names

Rule Example
Use lowercase with hyphens (kebab-case) frost-giant.svg
Frame numbers: two digits with hyphen walk-01.png, idle-02.svg
Retina assets: @2x suffix icon@2x.png
No spaces or special characters shadow-walker.svg (not shadow walker.svg)

Directory Structure

assets/ characters/ pantheon/ {character-name}/ portrait.png sprite-01.png sprite-02.png travelers/ wanderers/ shops/ enemies/ {enemy-name}.png enemies-svg/ {enemy-name}.svg items/ armor/ artifacts/ consumables/ materials/ quest/ weapons/ market/ {vendor-name}/ portrait.png idle-01.png walk-01.png ui/ dice/ chests/ currency/

Asset Dimensions

Asset Type Dimensions Format
Sprites (normal) 100x100px PNG, SVG
Sprites (4x source) 3840x3840px PNG
Portraits 240px, 120px, 60px, 30px PNG, SVG
Enemies 360x360px (source) PNG, SVG
Items 64x64px to 256x256px PNG, SVG
Icons 24x24px to 72x72px SVG
Dice 100x100px PNG, SVG (2-frame jitter)
Faction Icons ~44x44px (trimmed) SVG

Rarity System

Items and NPCs follow a 6-tier rarity system, each with a distinct color.

Tier Color Usage
Common #9e9e9e Basic items, common NPCs
Uncommon #4caf50 Slightly better than common
Rare #2196f3 Notable items, rare NPCs
Epic #9c27b0 Powerful items, elite NPCs
Legendary #ff9800 Top-tier items, boss NPCs
Unique #e91e63 One-of-a-kind special items

Vectorization Settings

PNG to SVG conversion using vtracer with ImageMagick preprocessing.

Standard Settings (150px)

# ImageMagick preprocessing magick input.png -resize "150x150" -fuzz "5%" +dither -colors 256 -depth 8 PNG8:temp.png # vtracer conversion vtracer --input temp.png --output output.svg \ --colormode color --hierarchical stacked --mode polygon \ --filter_speckle 1 --color_precision 6 --corner_threshold 60

Settings by Asset Type

Asset Type Size Colors Fuzz Corner
Enemies 150px 256 5% 60
Market Vendors 150px 256 5% 60
Flumes 150px 256 5% 60
Faction Icons native 32 2% 180 (sharp)
Sprites (normal) 100px 48 12% 120
Portraits varies 256 10% 180 (sharp)

Brand Rules

Do

  • Use "NEVER DIE GUY" as the brand name
  • Use dark mode exclusively
  • Follow the color palette strictly
  • Use kebab-case for file names
  • Include SVG variants for web assets
  • Maintain consistent spacing

Don't

  • Use separate taglines (brand name IS the tagline)
  • Use emojis in code, docs, or assets
  • Use light mode
  • Modify brand colors
  • Use spaces in file names
  • Skip vectorization for web assets

Trademark

NEVER DIE GUY is a trademarked brand name. It also serves as the tagline - no additional taglines are needed. Previous taglines like "data is eternal" or "death is essential" have been retired.
